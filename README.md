# hotconfig


## 使用方式
Example
请参考代码中测试用例

添加依赖
````
<dependency>
	<groupId>com.nosixtools</groupId>
	<artifactId>hotconfig</artifactId>
	<version>1.0.0</version>
</dependency>
````
需引入其它依赖支持
````

</dependency>

    <version>1.7.2</version>  
</dependency>  
<dependency>
    <groupId>commons-lang</groupId>

    <version>2.6</version>
</dependency>

</dependency>
````
